# 家庭收支记账软件 v2.0

## 项目简介
这是一个基于 Golang 开发的面向对象的家庭收支记账程序，无需连接数据库即可完成日常收支记录。

## 功能特性

### ✨ 新增功能（v2.0）
- 📊 **收支明细查询** - 查看所有收支记录，包含时间戳、类型、金额、余额、类别和说明
- 💰 **智能收入登记** - 支持多种收入类别（工资、奖金、投资、其他）
- 💸 **智能支出登记** - 支持多种支出类别（餐饮、交通、购物、娱乐、医疗、其他）
- 📈 **余额汇总** - 实时显示当前余额、总收入、总支出和交易笔数
- 📉 **分类统计** - 按类别统计收入和支出金额
- ⏰ **时间戳记录** - 每笔交易自动记录时间
- ✅ **输入验证** - 防止无效金额输入
- 🎨 **美化界面** - 更友好的用户交互界面

### 基本功能
1. 收支明细展示
2. 收入登记管理
3. 支出登记管理
4. 账户余额汇总
5. 分类统计分析
6. 安全退出系统

## 快速开始

### 环境要求
- Go 1.16 或更高版本

### 安装与运行

```bash
# 克隆项目
git clone https://github.com/Keviniscool-boy/-

# 进入项目目录
cd -

# 构建项目
go build -o familyaccount

# 运行程序
./familyaccount
```

## 使用说明

启动程序后，按照菜单提示选择相应的功能：

1. **收支明细** - 查看所有交易记录
2. **登记收入** - 记录收入，选择类别并添加说明
3. **登记支出** - 记录支出，选择类别并添加说明（会检查余额是否足够）
4. **余额汇总** - 查看账户总体情况
5. **分类统计** - 按类别查看收支统计
6. **退出系统** - 安全退出程序

## 技术特点

- 采用面向对象设计
- 使用结构体和方法封装功能
- 内存中存储交易记录（无需数据库）
- 完善的输入验证和错误处理
- 友好的中文用户界面
